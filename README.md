# 🔥 Fire Alarm System using Gas and Temperature Sensors


## 📘 Project Overview
This project is a fire alarm system that uses **gas** and **temperature** sensors to detect fire risks in environments such as homes, offices, and industries. It provides early warnings through a buzzer when gas leaks or elevated temperatures are detected.

## 🚨 Why Both Sensors?
Using both gas and temperature sensors ensures:
- **Early Detection**: Gas sensors can detect leaks or smoke before flames appear.
- **Heat Detection**: Temperature sensors alert even when there's no visible smoke.
- **Improved Accuracy**: Reduces false alarms by cross-verifying both indicators.
- **Enhanced Safety**: Increases response time to potential fire hazards.

## 🔧 Components Used
- MQ Gas Sensor
- LM35 Temperature Sensor
- Arduino UNO
- Buzzer
- Breadboard and connecting wires
- Power Supply

## ⚙️ Working Principle
1. The **gas sensor** detects smoke or gas leaks.
2. The **temperature sensor** monitors the ambient heat.
3. If any of the sensor values cross safety thresholds:
   - The **buzzer is activated**.
   - Optional: It can be extended to notify users via IoT or mobile alerts.
